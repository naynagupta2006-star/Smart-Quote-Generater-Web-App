# 💡 Smart Quote Generator

A responsive and interactive Quote Generator built using **HTML, CSS, and JavaScript**.  
This web application displays random inspirational quotes with author names and provides several useful features for users.

---

## 🚀 Features

- 🎲 Generate random quotes
- 😊 Mood-based quote filtering
- 📋 Copy quote to clipboard
- ✏️ Edit quote option
- 🔁 Auto generate quotes
- ⭐ Save favorite quotes
- 📤 Share quotes on social media
- 📱 Fully responsive design (mobile & desktop)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
